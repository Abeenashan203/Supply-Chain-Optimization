# ## Project Overview
# Features
A Python-based data cleaning and exploration pipeline built using Pandas to process agricultural 
and processing plant logistics data in Sri Lanka.
--Data Sanitization: Converts raw text logs (`cost_new.txt`) into a clean, structured tabular format (`cost_clean.csv`).
--Spatial Tracking: Coordinates plant and farm locations across regions like Gampaha, Kandy, and Matara.
--Exploratory Data Analysis (EDA): Baseline evaluation of production metrics such as productivity, operational costs, 
  fworkforce, distance, and wastage.
